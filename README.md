# ETH Trading Bot 🤖

Automatisierter Ethereum Trading Bot mit Machine Learning, Sentiment Analysis und Real-Time Dashboard.

## Features

- 🤖 **Machine Learning** - SGDClassifier für Trade-Predictions
- 📊 **Sentiment Analysis** - VADER + RSS News Analysis
- 🎯 **Risk Management** - Dynamische Position Sizing & Stop Loss
- 📈 **Real-Time Dashboard** - Premium Web Interface
- 📱 **Telegram Alerts** - Live Trade Notifications
- ⚡ **Adaptive Learning** - Auto-Parameter Optimization

## Performance Target

**Ziel**: 1% täglicher Gewinn (≈365% jährlich)

## Tech Stack

- Python 3.11
- FastAPI (Dashboard API)
- React + TypeScript (Frontend)
- Binance API
- scikit-learn (ML)
- NLTK (Sentiment)

## Deployment

- **Bot**: Railway
- **Dashboard**: Vercel
- **Database**: PostgreSQL (Railway)
