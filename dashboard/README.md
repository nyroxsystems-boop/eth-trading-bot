# ETH Trading Bot - Premium Dashboard

Modern, real-time trading dashboard for the ETH trading bot.

## Features

- 📊 Real-time trade monitoring
- 📈 Live performance metrics
- 🎯 Win rate & P&L tracking
- 🤖 ML confidence display
- 💹 Interactive charts
- 🔔 Live trade notifications
- 📱 Responsive design
- 🌙 Dark mode

## Tech Stack

- React 18 + TypeScript
- Vite
- TailwindCSS
- Framer Motion
- Recharts
- WebSocket

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy to Vercel:
```bash
vercel --prod
```

## Environment Variables

Create `.env`:
```
VITE_API_URL=https://your-railway-app.railway.app
VITE_WS_URL=wss://your-railway-app.railway.app/ws
```
